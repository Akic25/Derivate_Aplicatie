Project Name: Aplicatie pentru derivate 
Limbaj: C++
Aceasta apliatie a fost facuta impreuna cu un coleg de la facultate si este o aplicate stil academic care la introducerea unei expresii matematice iti arata derivata sa cu tot cu arborele acelei expresii. Functionalitatea acestui proiect sta la baza folosiri arborilor pentru descompunerea expresiilor.
